# Style AI Free PWA

Kostenlose, mobile WebApp ohne API-Kosten. Alles wird lokal im Browser gespeichert.

## Lokal starten

```bash
npm install
npm run dev
```

## Kostenlos hosten auf Vercel

1. Projekt zu GitHub hochladen
2. In Vercel importieren
3. Deploy klicken

Die App braucht kein Backend und keine Environment Variables.

## Handy Nutzung

Im Browser öffnen und über „Zum Home-Bildschirm“ als App speichern.
